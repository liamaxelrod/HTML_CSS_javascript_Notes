# How it works

1. **`fn main()`**
   * The `main` function is the entry point of every Rust program.
   * Code inside `main` runs when the program starts.
2. **`println!` Macro**
   * `println!` prints text to the console.
   * The `!` indicates that `println` is a macro, not a regular function.
   * Text inside quotes `"..."` is displayed exactly as written.
