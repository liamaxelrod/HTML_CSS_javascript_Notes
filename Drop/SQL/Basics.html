SQL Basics

SQL (Structured Query Language) is the standard language for working with relational databases.
It allows you to retrieve, manipulate, and manage data stored in tables. SQL is declarative,
meaning you specify what data you want, not how to fetch it. Key concepts include tables, columns, rows,
and data types, as well as core commands like SELECT, INSERT, UPDATE,
DELETE, and WHERE.

Think of a database as a cabinet: each table is a drawer, each row is a folder, and each column is a page
in the folder. Learning the basics of SQL queries, operators, and table structures is essential for
reading and managing data efficiently and safely.
