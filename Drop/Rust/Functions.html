Functions

Rust functions are blocks of code you can reuse. Each function can take inputs called parameters and return outputs. Functions have explicit types for inputs and outputs, which ensures clarity about what data enters and leaves the function.

Variables in Rust exist only within a defined scope. Functions, blocks, and loops create scopes. Rust also allows shadowing, letting you declare a new variable with the same name as an old one, replacing it safely. This helps transform values step by step without mutating the original.

Understanding functions, scope, and shadowing is key to writing code that is clear, predictable, and memory-safe. Rust encourages deliberate data flow: you always know what variables exist, where, and what values functions produce.
